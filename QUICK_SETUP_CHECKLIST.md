# ⚡ Quick Setup Checklist - Get Your Site Live in 30 Minutes

## Phase 1: Collections Setup (5 minutes)

Go to **Shopify Admin → Products → Collections**

### Create These 6 Collections:

**Collection 1:**
- [ ] Name: `Men's Fragrances`
- [ ] Handle: `mens-fragrances` (IMPORTANT: exact match)
- [ ] Save

**Collection 2:**
- [ ] Name: `Women's Fragrances`
- [ ] Handle: `womens-fragrances`
- [ ] Save

**Collection 3:**
- [ ] Name: `Unisex`
- [ ] Handle: `unisex`
- [ ] Save

**Collection 4:**
- [ ] Name: `Gift Sets`
- [ ] Handle: `gift-sets`
- [ ] Save

**Collection 5:**
- [ ] Name: `Best Sellers`
- [ ] Handle: `best-sellers`
- [ ] Type: **Automated**
- [ ] Condition: Product tag = `best-seller`
- [ ] Save

**Collection 6:**
- [ ] Name: `New Arrivals`
- [ ] Handle: `new-arrivals`
- [ ] Type: **Automated**
- [ ] Condition: Product tag = `new-arrival`
- [ ] Save

---

## Phase 2: Add Sample Products (15-20 minutes)

Go to **Shopify Admin → Products → Add product**

### For Each Product, You Need:

**Basic Info:**
- [ ] Product name (e.g., "Dior Sauvage Eau de Toilette 100ml")
- [ ] Description (copy from perfume brand website or write your own)
- [ ] Price (e.g., $89.99)
- [ ] Upload at least 1 product image

**Organization:**
- [ ] Check the appropriate **Collections**:
  - Add to `Men's Fragrances` OR `Women's Fragrances` OR `Unisex`
  - Add to `Gift Sets` if applicable
- [ ] Add **Tags**:
  - Type `best-seller` to add to Best Sellers collection
  - Type `new-arrival` to add to New Arrivals collection

**Example Products to Create:**

1. **Men's Perfume**
   - Title: "Creed Aventus Men's 100ml"
   - Price: $99.99
   - Collections: Men's Fragrances, Best Sellers
   - Tags: `best-seller`

2. **Women's Perfume**
   - Title: "Chanel Coco Mademoiselle 100ml"
   - Price: $109.99
   - Collections: Women's Fragrances, Best Sellers
   - Tags: `best-seller`

3. **Unisex Perfume**
   - Title: "Maison Margiela Beach Walk 100ml"
   - Price: $89.99
   - Collections: Unisex, New Arrivals
   - Tags: `new-arrival`

4-8. **Add 5-7 more products** across different categories

**Minimum for launch: 10 products**

---

## Phase 3: Upload Category Images (5 minutes)

Go to **Shopify Admin → Customize theme → Home page**

1. Scroll to **Featured Categories** section
2. For each category block:
   - Click the image placeholder
   - Upload a relevant image
   - Save

**Suggested images:**
- Men's: Luxury men's fragrances display
- Women's: Luxury women's fragrance display
- Unisex: Various fragrances
- Gift Sets: Perfume gift box sets

## ❗ Important Notes

- **Handles must be exact** (lowercase with dashes)
- **Use the Automated collection type** for Best Sellers and New Arrivals
- **Tag products properly** to auto-assign to collections
- **Add at least 1 image** per product
- **Save before moving to next section**

---

## 🆘 Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| Products not showing | Check collection handles match exactly |
| Best Sellers empty | Add `best-seller` tag to products |
| New Arrivals empty | Add `new-arrival` tag to products |
| Images not loading | Ensure images are uploaded to product |
| Category buttons disabled | Check collections exist with correct handles |
