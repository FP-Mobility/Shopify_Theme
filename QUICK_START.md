## 🆘 Troubleshooting

### Problem: Sections not showing on homepage
**Solution**: 
1. Check theme is published
2. Clear browser cache (Ctrl+Shift+Delete)
3. Hard refresh page (Ctrl+F5)

### Problem: Products not appearing in categories
**Solution**:
1. Make sure products are tagged correctly
2. Check collection has automated rules or manual products
3. Products must have category tag (mens, womens, etc.)

### Problem: Images not loading
**Solution**:
1. Check image file size (keep under 5MB)
2. Use common formats (JPG, PNG)
3. Verify image uploaded to Shopify, not external URL

### Problem: Metafields not showing on product page
**Solution**:
1. Verify namespace is "custom" (case-sensitive)
2. Ensure snippet is added to product template
3. Products must have metafield values filled in
4. Clear browser cache
