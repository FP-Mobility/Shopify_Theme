## 📞 Support & Questions

### Common Questions:

**Q: How do I preview changes before publishing?**
A: Use Shopify's Theme Customizer (Online Store > Themes > Customize) to see live changes before saving.

**Q: Can I change the colors?**
A: Yes! Edit CSS in section files or use theme customizer's color picker.

**Q: How many products do I need to launch?**
A: 10+ minimum recommended, but 20+ is better for a full experience.

**Q: Can I add more testimonials later?**
A: Yes! You can edit the testimonials section in theme customizer anytime.

**Q: Is the design mobile-friendly?**
A: Yes! All sections are fully responsive for mobile, tablet, desktop.
